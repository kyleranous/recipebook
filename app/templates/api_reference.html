End Points

Endpoint: api/recipes

Method: GET

What it does

Returns a list of all recipes in paginated form.

Message to send

  • page=[int]: Optional Page of the results. Default Value = 1
  • per_page=[int]: Optional Number of Results per Page. Default Value = 10, Max Value = 100
Example:
http://localhost:5000/api/recipes?page=1&per_page=5
The above request would send the first page of results, with 5 results per page.

Response

JSON Object:

  {
    "_links": {
      "next": "address to next page of results",
      "prev": "address to previous page of results",
      "self": "address to current page of results"
    },
    "_meta": {
      "page": [Current Page Number],
      "total_items": [Total Recipes in Query],
      "total_pages": [Total Pages in request]
    },
    "items": [
      {
        "id": [Recipe ID],
        "recipe_name": "Name of Recipe",
        "time": [Total Prep and Cook Time]
      }
    ]
  }
